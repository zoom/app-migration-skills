## Setup Instructions

### 1. Zoom Marketplace Configuration

1. Go to [Zoom Marketplace](https://marketplace.zoom.us/)
2. Click **Develop** → **Build App** → **Team Chat App**
3. Configure app settings:
   - **App Name:** {{APP_NAME}}
   - **Slash Command:** {{SLASH_COMMAND}}
   - **Bot Endpoint URL:** `https://your-domain.com/webhooks/zoom`
   - **Scopes:** `imchat:bot`
   - **Event Subscriptions:** `bot_notification`, `interactive_message_actions`

### 2. Environment Setup

```bash
# Install dependencies
npm install

# Copy and configure .env
cp .env.example .env
# Edit .env with your Zoom credentials
```

### 3. Run the App

```bash
# Development
npm run dev

# Production
npm run build && npm start
```
